## Description

Nemic Web Soluctions (NWS) OfficeWatch GraphQL API

Created with:
- [Nest](https://github.com/nestjs/nest) framework.
- Mongoose

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Checking the playgorund

Go to [http://localhost:3000/graphql](http://localhost:3000/graphql) while the app is running.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
